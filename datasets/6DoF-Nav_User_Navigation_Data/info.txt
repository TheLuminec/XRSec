Name: 6DoF-Nav User Navigation Data
Citation: T. R. Walters, L. J. P. van der Heijden, et al. 6DoF navigation dataset: head-mounted display user navigation data for volumetric videos. CWI 2023.

Sources:
 - https://www.dis.cwi.nl/6dof-nav-dataset/
 - https://github.com/cwi-dis/6DoF-HMD-UserNavigationData

Notes:
26 participants viewed dynamic point-cloud videos; dataset logs head position and rotation for each rendered frame.

Download Instructions:
Download the zip archive of the GitHub repository to obtain CSV files with viewport positions (HMDPX,HMDPY,HMDPZ) and rotations (HMDRX,HMDRY,HMDRZ) per frame.
